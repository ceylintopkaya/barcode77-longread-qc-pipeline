# 📥 Data Access and Setup

**⚠️ Note:** Due to upload issues with GitHub, the dataset could not be included directly in this repository. Please follow the steps below to access and prepare the data for analysis.

### 1. Download the Data
You can access all required files via the link below:
👉 [**Google Drive: Barcode77 Raw Dataset**](https://drive.google.com/drive/folders/1XJMNdBINgojbNKlKzYJNfrC3U-Noh2Dt)

### 2. Installation and Extraction
1. Download the dataset (ZIP) to your computer.
2. **Extract/Move** the `.fastq` files directly into the project's **root directory**.
3. Ensure that the files (e.g., .fastq) are located in the root directory and not hidden inside another subfolder.

> [!TIP]
> **Note:** When the analysis pipeline is executed, it will automatically create the necessary `data/` directory structure and organize your files. You do not need to create any folders manually.
